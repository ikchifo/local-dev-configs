# macOS Finder & File Picker Shortcuts (2026)

## Core Shortcuts

| Action | Shortcut |
|--------|----------|
| Copy Folder Path | ⌘ + ⌥ + C |
| Open "Go to Folder" | ⌘ + ⇧ + G |
| Show/Hide Path Bar | ⌘ + ⌥ + P |
| Paste Path in Picker | ⌘ + ⇧ + G, then ⌘ + V |

## Pasting a Path into a Browser File Picker

1. **Trigger Search:** Press ⌘ + ⇧ + G
2. **Paste:** Press ⌘ + V to insert your copied path
3. **Confirm:** Press Return to navigate to that folder

## Quick Tips

- **Drag-and-Drop:** Drag a folder icon directly from Finder onto any browser file picker window to instantly switch to that directory.

- **Proxy Icons:** Click and drag the small folder icon next to a window's title to move its path into a picker or terminal.

- **Root Access:** Type `/` while a file picker is active to automatically open the "Go to Folder" bar.
